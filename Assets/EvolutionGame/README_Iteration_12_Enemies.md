# README_Iteration_12_Enemies.md

## Что добавлено в Iteration 12

### Новые скрипты

**`BaseEnemy.cs`** — базовый класс для всех врагов. Обрабатывает:
- `HandlePlayerContact()`: поглощение если игрок больше на 10%, смерть если игрок меньше на 10%
- `SetEmissiveColor()`: URP emission с заданной интенсивностью
- `AnimateIntro()`: появление из нуля через DOTween OutBack
- `CanBeAbsorbed()`: виртуальный метод, переопределяется у BlackHole

**`EnemySpawnManager.cs`** — singleton, спавнит врагов по таблице весов с учётом текущей стадии эволюции. Новый враг появляется каждые 18 сек если на сцене меньше 4 врагов.

## Как тестировать

- **Sun**: появляется с орбитальными объектами вокруг — поглощай их, потом само Солнце если вырос
- **Black Hole**: не лезь пока маленький — затягивает и убивает. Появляется только на 4й стадии
- **Supernova**: видишь белый объект который краснеет — либо поглоти быстро, либо держись на расстоянии >12 единиц
- **Dark Matter**: трудно заметить, но гравитация тянет — следи за траекторией движения
- **Binary Pair**: два шара вращаются, у обоих своя коллизия — не врезайся если маленький

## Настройка EnemySpawnManager в инспекторе

- `Spawn Interval` — частота появления (дефолт 18 сек)
- `Max Enemies` — максимум врагов одновременно (дефолт 4)
- `Enemy Table` — веса и минимальные стадии для каждого типа
