# README_Iteration_03_GameFeel.md

## Что изменилось с Iteration 2

### Новые скрипты
- `CameraShake.cs` — singleton на Main Camera. `Shake(duration, strength, vibrato)`. Вызывается автоматически при смерти и при поглощении крупного объекта
- `AbsorptionEffect.cs` — singleton, ParticleSystem burst в точке поглощения. Размер и скорость частиц масштабируются от размера поглощённого объекта
- `PlayerTrail.cs` — контролирует ширину TrailRenderer в зависимости от скорости игрока. При движении trail широкий, при остановке сужается до нуля

### Изменённые скрипты
- `PlayerController.cs` — при поглощении вызывает `AbsorptionEffect.Play()` и лёгкий shake для крупных объектов; при смерти вызывает сильный `CameraShake`. Добавлен публичный метод `GetVelocityMagnitude()`
- `WorldObject.cs` — улучшена анимация всасывания: сначала объект чуть отскакивает от игрока (anticipation + squish), потом летит к нему и исчезает

### Editor скрипт
- `SetupGameScene_Iteration3.cs` — добавляет на Game сцену:
  - `CameraShake` компонент на Main Camera
  - `Trail` дочерний объект на Player с настроенным `TrailRenderer` и `PlayerTrail`
  - `AbsorptionEffect` объект со сконфигурированным `ParticleSystem`
  - Создаёт и назначает emissive материалы в `WorldObjectConfig` SO (синий для Small, оранжевый для Medium, красный для Large)

## Как тестировать

### Trail
- Начни двигаться — за игроком появляется синеватый след
- Останови игрока — след сужается и исчезает
- Чем быстрее движение — тем шире trail

### Absorption effect
- Поглоти маленький объект — burst мелких частиц в точке поглощения
- Поглоти средний объект — частицы крупнее + лёгкий shake камеры
- Объект сначала чуть отпрыгивает, потом стремительно летит к игроку

### Camera shake
- При смерти (столкновение с большим объектом) — заметный shake
- При поглощении объекта > 50% размера игрока — лёгкий shake

### Emissive объекты
- Малые объекты: синее свечение
- Средние: оранжевое свечение
- Крупные: красное свечение
- Bloom из Iteration 1 усилит эффект

## Ожидаемый результат
- Движение игрока читается сразу — виден trail
- Поглощение ощущается как удар, а не просто исчезновение объекта
- Смерть драматичная — shake + сжатие игрока
- Объекты визуально различимы по опасности через цвет свечения
